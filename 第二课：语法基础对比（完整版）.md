## 2.1 变量与数据类型 (25分钟)

### 2.1.1 变量声明与赋值

#### Python：动态类型

```
python复制# Python中变量无需声明类型，直接赋值即可
name = "张三"           # 字符串
age = 25               # 整数
height = 1.75          # 浮点数
is_student = True      # 布尔值

# 可以随时改变变量类型
x = 10                 # x是整数
x = "Hello"            # x变成字符串（完全合法）

# 多重赋值
a, b, c = 1, 2, 3
x = y = z = 0

# 交换变量（Python特色）
a, b = b, a
```

#### Java：静态类型

```
java复制// Java中必须先声明类型，再赋值
String name = "张三";
int age = 25;
double height = 1.75;
boolean isStudent = true;

// 类型一旦确定，不能改变
int x = 10;
// x = "Hello";  // ❌ 编译错误！类型不匹配

// 变量声明与赋值可以分开
int y;
y = 20;

// 多个同类型变量声明
int a = 1, b = 2, c = 3;

// 交换变量需要临时变量
int temp = a;
a = b;
b = temp;
```

#### 对比总结

| 特性         | Python      | Java         |
| ------------ | ----------- | ------------ |
| **类型声明** | 不需要      | 必须声明     |
| **类型检查** | 运行时      | 编译时       |
| **类型改变** | 允许        | 不允许       |
| **多重赋值** | 支持        | 不支持       |
| **变量交换** | a, b = b, a | 需要临时变量 |

        System.out.println("第10个斐波那契数: " + fibonacciRecursive(10));
    }
}

## 2.7 课程总结

### 核心知识点回顾

| 知识点         | Python特点       | Java特点           |
| -------------- | ---------------- | ------------------ |
| **类型系统**   | 动态类型         | 静态类型           |
| **变量声明**   | 不需要类型       | 必须声明类型       |
| **字符串**     | 单双引号通用     | 只能用双引号       |
| **缩进**       | 强制缩进（语法） | 缩进仅为规范       |
| **命名**       | snake_case       | camelCase          |
| **函数定义**   | def关键字        | 需要返回类型       |
| **多返回值**   | 直接返回元组     | 需要对象或数组     |
| **循环**       | for...in         | for/while/do-while |
| **列表推导式** | 支持             | 需要Stream API     |
